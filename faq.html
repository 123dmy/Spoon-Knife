Dynamic F.A.Q.
This FAQ add-on is created so you can have question and answers to some common queries about your product or service. Each link below dynamically opens up a hidden DIV area on this page. You can add more FAQ questions by copying and pasting one of the existing FAQ question DIV areas.

How do I edit these FAQ questions?
To edit this area, follow the steps on the help-dynamic-FAQ.html. Edit the new website page you created. Find the notes for "FAQ 1" in the page and edit the text in this area.
Can I add more FAQ questions on this page?
You can add as many FAQ questions to this webpage as your project requires. Follow these steps to add a new FAQ question.
  • Open the faq page you created and view the source code.
  • Find the text between the "<!-- FAQ 10 --> notes.
  • Copy this code and paste it below itself.
  • In the new pasted code, edit "faq10" in 2 places with "faq11".
  • Edit the "FAQ 10" notes in 2 places with "FAQ 11".
  • Open the "faq.htm" and test the FAQ links.
How do I change the FAQ answers area width?
Open the "FAQ_dynamic.js" javascript file with Notepad or any text editor. At the top of this file are options for the width of this answer area, for padding and for adding a border to this area. You may want to make a backup copy of this .js file before you make any edits to it.
How do I change the FAQ page colors?
The colors and fonts for the FAQ can be edited in the "FAQ.css" in the "FAQ" folder. The code for the text will be the "faqtext" class in your CSS file. Link colors are defined by the "link" classes near the top of the "FAQ.css" file. See the template colors support page for more help with changing template colors.
Can I add an image or video in an FAQ answer?
Yes, you can add images in an answer, or a video or a Flash animation object. Keep in mind that although the FAQ answers are hidden, they do still load when the FAQ page loads and can slow down the page if you add too many videos or larger sized images in an answer area.

Can I add links in the FAQ answers?
Yes. Links can be added to this area. You can also add new window links to any of the template pages or to other websites. Here is an example; Go to Allwebco Web Templates.
How much text can I include with each answer?
For each FAQ answer you can include as much or as little text as you would like. The answer area will expand to fit the text.
Can I add this FAQ to any website?
Yes. This FAQ addon and any Allwebco addon can be added to any website or to any website template. Step by step instructions are included. View Allwebco Add-ons.
Where are more Allwebco templates and addons?
Website templates and website addons are offered by Allwebco Design Corporation and are found only on the Allwebco website:

Click here to view Allwebco website addons.
Where is the addon help page?
The help page included in this addon download is named help-dropmenu.html.
