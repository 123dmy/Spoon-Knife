

The functions of various files are as under:

login.php -> Registers the user's data and uploads it to a remote database. Also has a login form for the user to log in to home.

home.php -> Here the user is welcomed and asked to upload his photo which is saved as <rollnum>.png in the local server.

upload.php -> Here the user can view the uploaded image and  logout. 

logout.php ->redirects the user back to the login page.

mydb.sql -> contains the commands for creating the remote database.
