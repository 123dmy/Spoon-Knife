      Steps for Git general:

  1. Fork the project To your GitRepository From the MasterRepository of the Project Location
  2. Clone the project from your GitRepository to your local WorkStation
  3. Open the project located in your local WorkStation via Editor
  4. Make changed to the work
  5. Commit it. New cahnges remains in your local repo
  6. Make MORE changed to the work
  7. Commit it. New cahnges STILL remains in your local repo
  8. NOW when you are ready to push your code
  9. Sync the code by clicking sync button
