Spark-Knife

Spark-Knife is a simple, yet powerful, tool for creating and managing your Spark projects.

What is Spark-Knife?

Spark-Knife is a simple, yet powerful, tool for creating and managing your Spark projects. It is a fork of the popular Yeoman project, but with a focus on Spark projects.

What is Spark?

Spark is a micro web framework for Java 8 with minimal effort. It is designed to make creating web applications easy.

What is Yeoman?

Yeoman is a generic scaffolding system allowing the creation of any kind of app. It allows for rapidly getting started on new projects and streamlines the maintenance of existing projects.

What is a generator?

A generator is a plugin that can be run with the spark-knife command to scaffold complete projects or useful parts.
